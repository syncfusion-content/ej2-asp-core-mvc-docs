# Overview of ASP.NET Core Extension for Visual Studio Code

The Syncfusion® ASP.NET Core Extension for Visual Studio Code simplifies using Syncfusion® ASP.NET Core components in an ASP.NET Core application by configuring the required Syncfusion® NPM or NuGet packages, namespaces, and themes for you.

The Syncfusion® ASP.NET Core Extension provides the following support in Visual Studio Code:

* [Download and Installation](download-and-installation.md): Install the Syncfusion® ASP.NET Core Extension from the Visual Studio Code marketplace.
* [Create Project](create-project.md): Create a Syncfusion® ASP.NET Core application (Syncfusion Web Application (Model-View-Controller), Syncfusion Angular, or Syncfusion React.js) with the configuration required to develop with Syncfusion® ASP.NET Core components.
